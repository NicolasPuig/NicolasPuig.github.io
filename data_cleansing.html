Data Cleansing

Basic Concepts of Data Preprocessing

Introduction

Data cleaning is the first step of data preprocessing, it consists in fixing or removing invalid and unwanted values in our dataset. Some common examples are:

  • Duplicates
  • Outliers
  • Missing values
  • Invalid formats
  • Corrupted data
In the following example we will be using RapidMiner Studio to easily clean our data, and the Titanic example dataset. Our goal is to predict if a passenger survived the sinking of the Titanic or not.

1. Analysing the dataset
